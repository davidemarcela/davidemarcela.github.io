# David ♥ Marcela

Um cantinho na internet feito com muito amor. 💕

Site romântico com mensagens de amor, contador de tempo juntos (desde **16/03/2025**),
galeria de fotos, linha do tempo e vários efeitos bonitos.

## Como ver o site

Abra o arquivo **`index.html`** no navegador (clique duas vezes nele).
É só isso — não precisa instalar nada.

## Como adicionar as nossas fotos 📸

Coloque as fotos na pasta **`assets/fotos/`** com os nomes
`foto1.jpg`, `foto2.jpg`, ... até `foto8.jpg`.

O site reconhece sozinho. Enquanto a foto não estiver lá, aparece um
cartãozinho de "em breve" no lugar. (Veja também `assets/fotos/LEIA-ME.txt`.)

## Música de fundo 🎵

Salve a música como **`assets/musica.mp3`** (exatamente esse nome).

A música começa a tocar **na primeira interação** da pessoa (um clique ou
ao rolar a página) — isso é uma regra dos navegadores, que bloqueiam som
automático antes de qualquer interação. O botão flutuante de nota musical
(canto inferior direito) liga e desliga a música quando quiser.

> Dica: o arquivo precisa ser **.mp3**. Se a sua música estiver em outro
> formato, dá pra converter pra mp3 em qualquer conversor online.

## Personalizar

| O que mudar | Onde |
|---|---|
| Mensagens de amor | `index.html` (seção "Mensagens") e mensagens do hero em `js/main.js` |
| Legendas / quantidade de fotos | lista `FOTOS` em `js/main.js` |
| Cores e visual | variáveis no topo de `css/style.css` |
| Data de início do namoro | `START_DATE` em `js/main.js` |

## Publicar no GitHub Pages

Como o repositório se chama `davidemarcela.github.io`, basta enviar (push)
estes arquivos para o GitHub que o site fica no ar em
`https://davidemarcela.github.io`.
